# CERVEJARIA-API: Projeto de Gerenciamento de Clientes, Vendas e Estoque de Cervejas

Este é um projeto de estudos que tem como objetivo criar um sistema para gerenciar clientes, vendas e o estoque de cervejas. O sistema foi desenvolvido como parte de um processo de aprendizado em programação e pode ser usado como ponto de partida para projetos mais complexos ou como um exercício de prática.

## Funcionalidades Principais

O sistema de gerenciamento de clientes, vendas e estoque de cervejas possui as seguintes funcionalidades principais:

- [X] **Cadastro de Clientes:** Permite registrar informações detalhadas sobre os clientes.

- [ ] **Cadastro de Cervejas:** Permite adicionar informações sobre as cervejas disponíveis no estoque, incluindo nome, tipo, teor alcoólico, quantidade disponível e preço unitário.

- [ ] **Registro de Vendas:** Possibilita registrar vendas, associando um cliente específico a uma ou mais cervejas vendidas. O sistema calcula automaticamente o valor total da venda e atualiza o estoque.

- [ ] **Consulta de Vendas:** Permite consultar o histórico de vendas, incluindo detalhes como data da venda, cliente, cervejas vendidas e valor total.

- [ ] **Gestão de Estoque:** Permite visualizar o estoque atual de cervejas e atualizar as informações, incluindo a adição de novos produtos, remoção de produtos esgotados e ajuste de quantidades.

- [ ] **Relatórios:** Gere relatórios que fornecem informações sobre vendas, clientes mais frequentes, cervejas mais vendidas e outras estatísticas relevantes.

### Dependências

- **Java 17:** A linguagem de programação utilizada para desenvolver o projeto.

- **MySQL 8.0.26:** Banco de dados utilizado para armazenar informações de clientes, cervejas e vendas.

- **Spring Framework 3.13:** Utilizado para criar a aplicação web.

- **Lombok:** Biblioteca utilizada para simplificar o código Java.
